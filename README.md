# MINaDAS — アメダス ミニお天気ダッシュボード

気象庁アメダスの公開データを使った、自分のエリア向けの簡易お天気ダッシュボードです（[Streamlit](https://streamlit.io/) 製）。APIキーは不要で、すべて気象庁・国土地理院の公開データを直接取得します。

## 機能

- **現在の観測値** … 気温・湿度・風速/風向・降水量と、簡易的な天気（晴れ/くもり/雨）
- **過去24時間の気温グラフ** … 10分間隔の観測から折れ線で表示（最低・最高も）
- **雨雲レーダー（降水ナウキャスト）** … 地理院地図に気象庁の高解像度降水ナウキャストを重ねて表示。公式ページをワンタップで開くボタン付き
- **ホームエリア** … 起動直後に自分のエリア（初期値：東京）を表示。検索で全国約1300地点に切替可能

## 必要なもの

- Windows（macOS/Linux でも動作します）
- Python 3.9 以上

## クイックスタート

```bash
git clone <このリポジトリのURL>
cd amedas_weather_app
pip install -r requirements.txt
streamlit run minadas.py
```

ブラウザで `http://localhost:8501` が自動で開きます。

## ワンクリック起動（Windows）

`アメダス天気_起動.bat` をダブルクリックするだけで起動します（初回は必要パッケージを自動インストール）。
黒いウィンドウがアプリ本体（サーバー）です。**閉じると停止**します。

## ホームエリアの変更

起動後に画面上部の検索から選ぶか、`minadas.py` 冒頭の次の1行を書き換えてください。

```python
HOME_STATION_CODE = "44132"  # 東京
# 例) 横浜=46106 / 千葉=45212 / 熊谷=43056 / 銚子=45148
```

観測所番号は [気象庁アメダス](https://www.jma.go.jp/bosai/amedas/) で確認できます。

## データ出典・免責

- 気象庁（アメダス観測データ／高解像度降水ナウキャスト）
- 国土地理院（地理院タイル）

本アプリの「天気（簡易判定）」は降水量・日照から機械的に推定した目安であり、気象庁の予報・実況とは異なる場合があります。個人利用向けの簡易ツールです。
